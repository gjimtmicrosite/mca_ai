💻 Project 1: Algorithm Optimization Study Google Weeks 1–4

Objective: Compare time and space complexity of alternative algorithms for the same problem.

AI Tools: ChatGPT (complexity explanation), GitHub Copilot (refactoring suggestions)

🟠 Phase 1 — Manual Understanding
  • Understand the problem / system context without using AI as a shortcut.
  • Write inputs, outputs, constraints, assumptions and expected result.
  • Create Google Drive folder: [UserID]_P1_Algorithm_Optimization_Study.
🔵 Phase 2 — Design and Implementation
  • Prepare algorithm, pseudocode, flow or module plan.
  • Implement code manually first, then use AI only for explanation/refactoring support.
  • Record test cases and screenshots.
🟢 Phase 3 — AI-Assisted Review
  • Use ChatGPT/Gemini/Copilot to validate logic and improve clarity.
  • Compare AI suggestions with classroom concepts.
  • Document what was accepted, rejected and corrected.
🔵 Phase 4 — Report + Viva
  • Prepare final report, code/output screenshots, AI reflection and viva notes.
  • Upload final files through the submission guide.

Final Deliverables

Code FilesClean source code with comments and proper naming
Testing EvidenceTest cases, screenshots, error corrections and performance table where needed
