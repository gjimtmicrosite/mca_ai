🗂️ Weekly Session Guide — Hour-by-Hour Plan

Important: MCA Semester 1 has no certification/mock-test component for students. This guide is built around exactly 3 Shadow Projects: P1 Weeks 1–4, P2 Weeks 5–8, P3 Weeks 9–12.

3 Hours · MCA Sem 1
Time BlockFaculty ActionStudent Activity

🎙️ Concept + Demo

  • Select problem, create folder, run baseline code
